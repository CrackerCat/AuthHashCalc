This directory contains Portable Executable test file collections.

[bin] - malformed pe file used for testing
[corkami_pe_bin.7z] - 7z of unmodified collection archive of PE format files taken from https://github.com/corkami/pocs/tree/master/PE
[waleedassar_pe_bin.7z] - 7z unmodified collection archive of PE format files taken from waleedassar blog

WARNING: use them with caution, some of they may crash OS.
